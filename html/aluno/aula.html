RESUMO

Nesta aula, você aprenderá sobre operadores lógicos no VisualG, que são utilizados para combinar expressões booleanas (verdadeiro ou falso).

  1. 1. Operadores Lógicos no VisualG

    No VisualG, temos três operadores principais:

    • E (e) → Retorna verdadeiro apenas se ambas as expressões forem verdadeiras.
      • Exemplo: verdadeiro e falso → falso
    • OU (ou) → Retorna verdadeiro se pelo menos uma das expressões for verdadeira.
      • Exemplo: verdadeiro ou falso → verdadeiro
    • NAO (nao) → Inverte o valor da expressão.
      • Exemplo: não verdadeiro → falso
  2. 2. Ordem de Precedência

    A ordem de execução dos operadores segue a seguinte prioridade:

    1. NAO (nao)
    2. E (e)
    3. OU (ou)

    Se necessário, podemos usar parênteses para modificar a ordem de avaliação.

  3. 3. Aplicação das Leis de De Morgan

    As Leis de De Morgan ajudam a transformar expressões lógicas:

    ¬(X∧Y) = ¬X∨¬Y

    ¬(X∨Y) = ¬X∧¬Y
